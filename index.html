GIT Cheatsheet
Commands
Notes
git help
invokes git manual to terminal display
git init
initializes git version control for a project
git add .
adds all (.) file changes to git
git commit
commits changes to git
git log
invokes git log
git log --grep="search term"
searches for a term inside the log. grep stands for "global regular expression"
git status
displays current status of git project, including "untracked" files, "files to be committed" (or staged), etc.
git diff
displays changes to working directory files. compares changes to file of the working directory copy against the staging directory copy
git diff --staged
displays changes to files in the staging area only. compares changes to file of the staging directory copy against the repository copy
git mv fileName.txt newFileName.txt
renames file from command line
