GIT Cheatsheet
Commands
Notes
git help
invokes git manual to terminal display
git init
initializes git version control for a project. run this command from inside root of project directory
git add .
adds project changes to git. run this command from inside root of project directory
git commit
commits changes to git. run this command from inside root of project directory
git log
invokes git log. run this command from inside root of project directory
